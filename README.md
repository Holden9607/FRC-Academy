A student-made, basic FRC training tool for learning motor logic and button mapping for **REV (NEO)** and **CTRE (Kraken)** hardware.

---

> [!CAUTION]
> ### Required Setup Before Running
> Because **FRC Academy** runs and compiles real robot code logic locally on your machine, you **must** have Java installed on your computer, or the application will not open!
> 
> * **If using FRC Academy v1.1.0 or Lower:** You must have at least **Java 8** installed.
> * **If using FRC Academy v2.0.0 or Up:** You must have at least **Java 21** installed.

---

## 📥 Getting Started

1. **Download:** Click the badge above to download the latest `.zip` release.
2. **Extract:** Right-click the downloaded file and select **"Extract All"**.
3. **Run:** Open the extracted folder and double-click `FRC-Academy.exe` to begin training.

> ⚠️ **Note on Antivirus:** Most antivirus software will likely scan the `.exe` upon opening but it is completely safe!

---

## 🛠️ Training Features
* **NEO (REV) Training:** Practice SparkMax motor control and basic button mapping.
* **KRAKEN (CTRE) Training:** Learn the Phoenix 6 syntax and DutyCycleOut logic.
* **Empty-Editor Logic:** Manual code entry to build muscle memory for competition.
